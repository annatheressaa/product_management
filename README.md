# NexusSupply | Product Management System

A professional Python-based desktop application for managing industrial tool inventory.

Key Database Features
This project demonstrates advanced SQL and Database Management concepts:
- **Data Operations**: Full CRUD (Create, Read, Update, Delete) via the UI.
- **Relational Data**: Uses **JOIN queries** to connect Orders, Products, and Items.
- **Summarized Reporting**: Uses **Aggregate Functions** (SUM, COUNT) and **MySQL Views**.
- **Data Integrity**: Enforces **Foreign Key Constraints** and data validation.
- **Automation**: Implements **Triggers** for automatic stock deduction.

Tech Stack
- **Frontend**: CustomTkinter (Modern Python GUI)
- **Backend**: Python 3.x
- **Database**: MySQL (XAMPP/phpMyAdmin)

How to Run
1. Import the provided SQL file into your phpMyAdmin.
2. Ensure XAMPP MySQL is running.
3. Install dependencies: `pip install customtkinter mysql-connector-python`
